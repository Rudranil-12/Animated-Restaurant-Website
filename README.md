# 🍽️ Foodies:– Modern Animated Restaurant Website

A modern, fully responsive and animated restaurant landing page built using **ReactJS**, **Tailwind CSS**, and **Framer Motion**.

This project focuses on clean UI design, smooth animations, and modern layout structure suitable for restaurant or food-based businesses.

---

## ✨ Features

- ⚡ Modern & Clean UI Design
- 🎨 Fully Responsive Layout
- 🎬 Smooth Animations using Framer Motion
- 🌿 Tailwind CSS Styling
- 🛒 Interactive Navigation Bar
- 🍰 Popular Recipes Section
- 📱 Mobile Friendly
- 🎯 Reusable Components Structure

---

## 🛠️ Tech Stack

- ReactJS
- Tailwind CSS
- Framer Motion
- React Icons
- Vite (if you used Vite)

---

## 🚀 Getting Started

### 1️⃣ Clone the Repository

git clone https://github.com/Rudranil-12/Animated-Restaurant-Website.git

### 2️⃣ Navigate to Project Folder

cd Animated-Restaurant-Website

### 3️⃣ Install Dependencies

npm install

### 4️⃣ Run Development Server

npm run dev

---

## 📦 Framer Motion Installation

npm install motion

## React Icons Installation
